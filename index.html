Einstufungs­test Deutsch

Dieser Test erlaubt die Einschätzung Ihrer Sprachkenntnisse basierend auf dem Gemeinsamen Europäischen Referenzrahmen für Sprachen. Im Anschluss an den Test können Sie Ihre Ergebnisse an die Plus Training OG weiterleiten, um den passenden Sprachkurs für Sie zu finden. Bitte beachten Sie, dass dieser Test nur einer ersten Einstufung dient und nicht als offizielle und/oder endgültige Bestätigung Ihres Sprachniveaus gelten kann.

This test allows you to find out your language skill level based on the Common European Reference for Languages. Once you’ve answered all the questions, you can send your results to "Plus Training OG" to help you find the right language class for you. Please note that this test is only for the purpose of initial placement and cannot serve as an official and/or final confirmation of your language skill level.

Anleitung | Instructions

Beantworten Sie die Fragen, indem Sie abhängig von der Aufgabe die richtige Antwortmöglichkeit auswählen. Wenn Sie etwas nicht wissen, wählen Sie „Ich weiß nicht.“ Welche Fragen Sie angezeigt bekommen, hängt von Ihren Antworten ab. Vergessen Sie nicht, die Aufgabenstellung bei jeder Frage sorgfältig zu lesen. Dieser Test beinhaltet keine Höraufgaben.

Answer the questions by choosing the right options. If you don’t know the answer, choose ‘Ich weiß nicht.’ Which questions you see depends on the answers you give. Don’t forget to read each task’s instructions carefully so that you know what you need to do. This test does not include any listening tasks.
